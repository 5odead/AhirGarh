# Ahirgarh — Community Heritage Wiki

A Wikipedia-style heritage portal documenting the history, gotras, villages and personalities of the Ahir/Yadav community of India.

## Live Site
🌐 [ahirgarh.vercel.app](https://ahirgarh.vercel.app)

## Built With
- React + TypeScript
- Tailwind CSS
- Framer Motion
- Fuse.js
- Vite
- Vercel

## Features
- Community wiki with articles, gotras, villages and personalities
- Client-side fuzzy search with CMD+K shortcut
- Dark and light mode
- Fully static — no backend, no database, no login

## Disclaimer
Ahirgarh is a cultural documentation project built for educational and heritage preservation purposes only.
